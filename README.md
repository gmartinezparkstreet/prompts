## PR review prompts

This repository contains reusable prompt templates for code and pull request reviews.

- **Angular PR review prompt**: General Angular application code (components, services, modules, templates).  
  - [View prompt](pr-reviews/angular.md)

- **Angular unit test PR review prompt**: Focused only on Angular unit test files (`*.spec.ts`).  
  - [View prompt](pr-reviews/angular-tests.md)
